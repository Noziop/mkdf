NOM
       mkdf - MaKeDir&Files, utilitaire pour créer des structures de répertoires et fichiers

SYNOPSIS
       mkdf [OPTIONS] [EXPRESSION]

DESCRIPTION
       mkdf est un utilitaire pour créer rapidement des structures de répertoires et de fichiers en utilisant une syntaxe d'expansion d'accolades ou des modèles prédéfinis.

       Si lancé sans arguments, mkdf démarre en mode interactif.

OPTIONS
       -h, --help
              Affiche l'aide et quitte.

       -v, --verbose
              Mode verbeux. Affiche les détails de chaque opération.

       -q, --quiet
              Mode silencieux. Supprime tous les messages de sortie.

       -s, --simulate
              Mode simulation. N'effectue pas réellement les créations.

       -f, --force
              Force la création même si les répertoires existent déjà.

       -i, --interactive
              Lance le mode interactif de MKDF.

       -w, --web
              Lance l'interface web de MKDF.

       -p, --port=NUMBER
              Spécifie le port pour l'interface web (défaut: 8080).

       -c, --create=PROJECTTEMPLATE
              Crée un nouveau projet basé sur un modèle prédéfini.

       --version
              Affiche la version de MKDF et quitte.

SYNTAXE D'EXPANSION
       MKDF utilise une syntaxe d'expansion avec des accolades pour définir efficacement des structures complexes :

       - {a,b,c} crée a, b et c au même niveau

       - Les chemins terminés par / sont créés comme des répertoires

       - Les autres sont créés comme des fichiers vides

       - Vous pouvez imbriquer les accolades : {a/{b,c},d} crée a/b, a/c et d

MODÈLES DE PROJET
       Les modèles de projet suivants sont disponibles :

       simple Structure minimaliste adaptée aux petits projets.

       multi  Structure plus complète avec séparation des composants logiques.

       docker Structure incluant une configuration Docker.

       fastapi
              Structure complète pour une application backend basée sur FastAPI.

       vue-vite
              Configuration pour un projet frontend avec Vue.js et Vite.

       flask  Structure pour une application web Python avec Flask.

       react  Configuration pour une application frontend React.

EXEMPLES
       Créer une structure simple :
              mkdf 'projet/{src/,docs/,tests/}'

       Créer une structure plus complexe avec imbrication :
              mkdf 'projet/{src/{models/,views/,controllers/},docs/,tests/{unit/,integration/}}'

       Créer des fichiers et des répertoires :
              mkdf 'projet/{README.md,src/,docs/USAGE.md}'

       Créer un projet à partir d'un modèle :
              mkdf -c mon-app react

       Lancer l'interface web :
              mkdf --web

       Lancer l'interface web sur un port spécifique :
              mkdf --web --port 3000

FICHIERS
       ~/.config/mkdf/config.ini
              Fichier de configuration utilisateur.
